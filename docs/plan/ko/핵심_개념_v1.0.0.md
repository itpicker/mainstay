# Mainstay 주요 개념: 자율형 전문가 협업
- **버전**: 1.0.0
- **작성 시간**: 2025-12-27 16:57

Mainstay는 사용자가 전문 AI 에이전트를 "채용"하고, 이들이 복잡한 프로젝트 목표를 달성하기 위해 자율적으로 협업하는 플랫폼입니다. 단순한 도구 호출을 넘어 인간과 유사한 전문적 협업 모델을 지향합니다.

## 1. "채용(Hiring)" 모델
- 사용자는 AI 에이전트를 고수준 전문가(예: 마케팅 디렉터, 시니어 개발자, 법률 전문가)로 대우합니다.
- 각 에이전트는 고유한 전문 분야, 도구 세트 및 전문적인 "철학"을 가지고 있습니다.
- 에이전트를 채용한다는 것은 프로젝트의 특정 부분을 관리할 수 있는 자율성을 부여함을 의미합니다.

## 2. 자율적 프로세스 관리
- 고정된 워크플로우와 달리, Mainstay 에이전트는 **스스로 프로세스를 정의**합니다.
- 사용자의 프로젝트 요청에 따라 전문가들이 협업하여 일련의 작업 순서를 수립합니다.
- 시스템은 자가 수정 기능을 갖추고 있습니다. 작업 단계가 실패하면 에이전트는 사용자의 미세 관리 없이 계획을 재평가하고 조정합니다.

## 3. 전문가 팀 시너지
- 에이전트들은 의존성을 해결하기 위해 서로 소통합니다 (예: 개발자 에이전트가 리서처에게 API 문서를 요청함).
- 공유된 프로젝트 컨텍스트: 모든 에이전트가 동일한 프로젝트 보드와 이력을 공유하여 정렬을 유지합니다.

## 4. 기능 확장 (Meta-Tooling)
- 에이전트는 초기 도구 세트에 국한되지 않습니다.
- 작업에 특정 기능(예: 커스텀 데이터 파서)이 필요한 경우, 에이전트는 작업을 완료하기 위해 스스로 **도구를 설계하고 구축**할 수 있습니다.
